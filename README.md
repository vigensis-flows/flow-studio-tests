# Flow Studio Tests

This is a Flow Studio test workspace.

## Directory Structure

- `docs/` - All documentation and deliverables
  - `product/` - Product vision, strategy, and roadmap
  - `discovery/` - Research, experiments, and market analysis
  - `design/` - Design patterns, decisions, and specifications
  - `architecture/` - System architecture and technical decisions
  - `process/` - Development methodology and workflows
  - `reference/` - Style guides and reference materials
  - `operations/` - Deployment, monitoring, and runbooks
  - `templates/` - Document templates
- `.claude/` - AI agent configuration
